# Pomodoro Timer

This project is intended to create an arduino controlled timer which implements the pomodoro technique, to help with studying. Achieving this will require electrical, software and mechanical design, the requirements for each of which are listed below.

##  Requirements - Electrical
- The device shall run on an Arduino Nano microcontroller and utilise no more GPIO pins than are available to it
- The device shall contain a seven segment display capable of displaying a countdown
- The device shall contain an LED to visually alert a user when the timer expires
- The device shall contain a buzzer to auditorily alert a user when a timer expires
- The device shall contain a potentiometer capable of controlling the volume of said buzzer
- The device shall contain a vibration device to haptically alert a user when a timer expires
- The device shall contain switches to enable or disable each of these feedback devices; these switches shall be associated with an LED to indicate their state
- The device shall contain a "Begin" button and a "Reset" button to start the timer and stop timer or disable alert, respectively
- The device may, depending on the availability of GPIO pins, be capable of using extra buttons for increased functionality
- The device shall be powered via commonly available household batteries, such as AA or AAA
